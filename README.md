# analysis_of_online_store_activity

В данной работе проводилось исследование продаж компании, с целью выяснить в какой регион и в какой рекламный канал
инвестировать больше денег на привлечение клиентов.

Была проведена предобработка данных:
1) удалены пропуски
2) названия столбцов приведены к стандарту PEP8
3) удалены дубликаты
4) приведены типы данных
5) проведена обработка выбросов

Проведен исследовательский анализ данных для понимания их особенностей:
1) определен исследуемый период
2) проведен графический и аналитический анализ
3) проведены математический расчеты, характеризующие данные

Проведена проверка гипотез с использованием различных методов
для соотвествующих типов данных:
1) критерий Краскела-Уоллиса для категориальной шкалы (кол-во уровней >= 3) и количественной (ненормальное распределение)
2) корреляция Спирмена для количественных шкал с распределениями, отличными от нормального
3) двухвыборочный z-test для пропорций (2 бинарных признака)

Проведено прогнозирование дальнейших продаж с применением линейной регрессии 
(для прогноза дохода и для прогноза числа покупателей):
1) подготовлены данные для модели (кодировка через OneHotEncoder)
2) предсказан прогноз на год по региону, рекламному каналу и бюджету
3) определены лучшие сегменты по доходу и числу покупателей
4) оценена кросс-канальная стратегия для двух перспективных сегментов с суммированием прогнозов.
